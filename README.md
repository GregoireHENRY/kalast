## Usage

Add the dependency **kalast** to your `Cargo.toml`:

```toml
[dependencies]
kalast = "0.3"
```

Add to your code:

```rust
use kalast::prelude::*;
```

and refer to [kalast::prelude online][docsrs-prelude] for available modules and items.
